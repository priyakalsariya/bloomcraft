# 🌸 Bloom Craft

Bloom Craft is an AI-powered bouquet customization web application built using Django.  
Users can design personalized bouquets by selecting flowers, leaves, accessories, and wrapping papers for different occasions like birthdays, anniversaries, weddings, and more.

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 🌷 Bouquet Customization
- 🎉 Occasion-Based Selection
- 🛒 Shopping Cart System
- 💳 Checkout Functionality
- 📦 Order Tracking
- 🧾 Invoice Download
- 🖼️ Image Upload & Media Handling
- 📱 Responsive UI Design
